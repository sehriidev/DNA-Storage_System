# DNA-Based Digital Data Storage System

This project implements a DNA-Based Digital Storage System using Python.

The system converts digital image and text data into DNA nucleotide sequences through DNA encoding techniques and reconstructs the original information using DNA decoding methods.

## Features

* Image Encoding and Decoding
* Text Encoding and Decoding
* DNA Storage Simulation
* DNA Corruption Simulation
* Storage Accuracy Analysis

## Technologies Used

* Python
* NumPy
* Pillow (PIL)
* Matplotlib

## Project Structure

* input_data/
* scripts/
* dna_storage/
* output/
